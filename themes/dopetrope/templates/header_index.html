Build

μbrew provides you with everything you need to build your own game console. We offer easy-to-follow tutorials that empower you to construct an open and customized gaming machine within a few hours. Learn how a modern, embedded operating system and its software stack works!

Learn

The μbrew gaming platform supports development with the state-of-the-art V-Play Game Engine. Learn how to build your own games with V-Play, and publish your game on μbrew. You will also learn how to build your game for the iOS and Android platforms, to reach a wider audience and make money!

Play

And after all the fun to build your own game console don't forget to play! The basic installation comes with open source games like Flappy Bird. Visit the μbrew repository to find more games and play the games with your friends. Our open platform will provide you with new games for endless fun!
